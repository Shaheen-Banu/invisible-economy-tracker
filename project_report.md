## Objective
The project estimates unrecorded student spending using behavioral indicators such as cash usage, expense tracking habits, bill splitting behavior, and forgetfulness in recording expenses.

## Dataset
Collected using Google Forms from college students.

## Techniques Used
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Regression Modeling
- KMeans Clustering
- Streamlit App Development

## Key Insights
- Students using cash frequently show higher hidden spending behavior.
- Hostel students tend to have more untracked expenses.
- Students who forget recording expenses frequently show impulsive behavior patterns.

## Models Used
- Linear Regression
- KMeans Clustering

## Conclusion
The project successfully reconstructs invisible spending behavior using indirect behavioral indicators.
