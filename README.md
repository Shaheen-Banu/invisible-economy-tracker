# Invisible Economy Tracker

## 📌 Project Description
This project analyzes student spending behavior to identify hidden financial patterns.

## 🎯 Objective
- Track spending habits
- Identify untracked expenses
- Classify financial risk levels

## 🛠 Tools Used
- Python
- Pandas
- Data Analysis

## 📊 Features
- Spending Ratio Calculation
- Tracking Categorization
- Risk Level Prediction

## 🚀 How to Run
1. Open terminal
2. Navigate to project folder
3. Run: python -m app.main
